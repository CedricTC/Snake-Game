# Yılan Oyunu

Bu Python projesi, klasik yılan oyununu basit bir grafiksel arayüzle uygular. Oyuncu, yılanı yönlendirerek ekrandaki yemeği yemeye çalışırken, yılanın kendi bedenine veya duvarlara çarpmasını önlemelidir.

## Kurulum

1. Projeyi bilgisayarınıza indirin.
2. Python yüklü değilse, indirip kurun.
3. Projenin bulunduğu dizine gidin.
4. Terminal veya komut istemcisinde `python snake_game.py` komutunu çalıştırarak oyunu başlatın.

## Nasıl Oynanır

- Yılanın başını yön tuşları (Yukarı, Aşağı, Sağ, Sol) ile yönlendirin.
- Yılan yemeğe ulaştığında uzar ve puan kazanırsınız.
- Duvarlara veya kendi bedeninize çarptığınızda oyun sona erer.

## Kontroller

- Yukarı: Yukarı ok tuşu
- Aşağı: Aşağı ok tuşu
- Sağa: Sağ ok tuşu
- Sola: Sol ok tuşu

## Özellikler

- Oyun sırasında toplanan puanlar ekranda gösterilir.
- Yılanın hızı oyun ilerledikçe artar.
- Oyunun kaydedilmesi veya yüklenmesi gibi ileri özellikler ekleyebilirsiniz.
